⚠️ Safety Measures and ESD Protection

What is ESD?

Electrostatic Discharge (ESD) is the sudden flow of electricity between two objects caused by contact or electrical short. Even a small discharge (invisible to the human eye) can permanently damage sensitive computer components!

Safety Best Practices:

  • ⚡ Work on a clean, non-conductive surface
  • 👕 Avoid synthetic clothing (prefer cotton)
  • 🔌 Use an anti-static wrist strap and ground yourself
  • 🔋 Unplug power cable and switch PSU OFF before starting
  • ✋ Handle components by edges only, never touch pins or contacts

🔓 PC Disassembly Step-by-Step

Follow these steps in order to safely disassemble a PC:

1 Power Off & Disconnect - Turn off the PC, unplug all cables, switch PSU OFF
🔌❌
2 Remove Case Panels - Remove side panels using screws (store safely)
🗄️
3 Disconnect Internal Cables - Remove power, SATA, and front panel connectors
🔗
4 Remove GPU - Unscrew bracket, release PCIe latch, pull straight out
🎮
5 Remove RAM - Push down clips on both sides, pull out gently
💾
6 Remove CPU Cooler - Disconnect fan cable, unscrew mounting (be careful!)
❄️
7 Remove CPU - Lift retention arm, carefully remove CPU (don't touch pins!)
🧠
8 Remove Storage Drives - Unscrew SSDs/HDDs from mounting brackets
💿
9 Remove Motherboard - Unscrew all mounting screws, lift out carefully
🏗️
10 Remove PSU - Unscrew from back, disconnect modular cables, remove
⚡

🔨 PC Assembly Procedure

Assembly Order:

1 Install CPU - Align triangle markers, place gently, secure retention arm
2 Apply Thermal Paste - Small pea-sized dot in center of CPU
3 Install CPU Cooler - Mount securely, connect fan to motherboard
4 Install RAM - Align notch, press down until clips lock
5 Mount Motherboard - Install I/O shield, align standoffs, screw in
6 Install PSU - Fan facing down/out, secure with screws
7 Connect Power Cables - 24-pin ATX, 8-pin CPU, PCIe if needed
8 Install Storage - Mount drives, connect SATA/NVMe
9 Install GPU - Insert into top PCIe x16 slot, secure bracket
10 Cable Management - Route cables behind motherboard tray, ensure good airflow

⚙️ Exploring BIOS / UEFI

How to Access BIOS:

How to Enter BIOS:

Restart your computer and press DEL or F2 key repeatedly during boot (before Windows logo appears). Some manufacturers use F1, F10, or ESC.

What to Check:

  • 🌡️ CPU Temperature - Should be 30-50°C at idle
  • 💾 RAM Size - Verify all installed memory is detected
  • 📀 Boot Order - Set priority (SSD first, USB for OS install)
  • 💿 Storage Drives - Check if all drives are detected
  • 🔄 Firmware Version - Note BIOS/UEFI version
Lab Task: Each student will document their system specifications from BIOS and compare results with classmates.

💿 Installing an Operating System

Installation Steps:

1 Create Bootable USB - Use Rufus/Media Creation Tool to create installation media
2 Set Boot Device - Enter BIOS, change boot order to USB first
3 Boot from USB - Restart PC, boot from USB drive
4 Format Drive - Delete old partitions, create new ones
5 Install OS - Follow installation wizard, enter product key
6 Install Drivers - Download and install motherboard, GPU drivers
7 Configure Windows - Complete setup, activate, and update
🪟
🐧
💡 Performance Tip:

SSDs are 10-50x faster than HDDs for OS installation. A Windows install takes ~10 minutes on SSD vs 30+ minutes on HDD!

🔧 Hardware Troubleshooting Scenarios

Click each scenario to reveal the solution!

⚡ Scenario 1: PC Won't Power On At All
🔍 Diagnosis Steps:
1. Check if PSU switch is ON (back of PSU)
2. Verify power cable is firmly connected
3. Test power outlet with another device
4. Check if PSU cables are connected to motherboard (24-pin + 8-pin CPU)
5. Try paperclip test on PSU to verify it works
💡 Common causes: Dead PSU, loose cables, faulty power button
🖥️ Scenario 2: Fans Spin But No Display
🔍 Diagnosis Steps:
1. Reseat RAM modules (remove and reinstall firmly)
2. Try single RAM stick in different slots
3. Check monitor cable connection (try different cable)
4. Verify GPU is fully seated in PCIe slot
5. Connect monitor to motherboard (if integrated GPU available)
6. Clear CMOS (remove battery for 30 seconds)
💡 Common causes: Loose RAM, faulty GPU, monitor/cable issue
🔥 Scenario 3: System Shuts Down Randomly
🔍 Diagnosis Steps:
1. Check CPU temperature in BIOS (should be under 80°C)
2. Verify CPU cooler is properly mounted and fan spinning
3. Check if thermal paste has dried out
4. Test PSU with multimeter or swap with known good unit
5. Monitor temperatures with HWMonitor during stress test
6. Check Event Viewer for error logs
💡 Common causes: Overheating CPU, insufficient/failing PSU, dust buildup
💾 Scenario 4: Storage Drive Not Detected
🔍 Diagnosis Steps:
1. Check SATA/power cables are firmly connected
2. Try different SATA port on motherboard
3. Verify drive appears in BIOS/UEFI
4. For NVMe: ensure M.2 screw is tight and drive fully inserted
5. Test drive in another computer or external enclosure
6. Initialize disk in Disk Management (Windows)
💡 Common causes: Loose cables, dead drive, needs initialization

💻 Hardware Knowledge for Programmers

Why understanding hardware makes you a better programmer!

🧠

CPU Cores & Threads

Parallel Programming

What to check:
• Number of cores (4, 6, 8+)
• Thread count (with SMT/HT)
• Clock speed (GHz)

Why it matters:
Multi-threaded apps run faster with more cores. Use Task Manager to see core usage during compilation or data processing.
🎮

GPU VRAM

AI & Graphics

What to check:
• VRAM size (4GB, 8GB, 12GB+)
• GPU model and compute capability
• CUDA cores (NVIDIA) or Stream Processors (AMD)

Why it matters:
AI models need sufficient VRAM. Small models: 4-6GB, Large models: 12GB+. Check before training!
💾

RAM Capacity

Data Processing

What to check:
• Total RAM installed
• RAM speed (MHz)
• Available vs Used memory

Why it matters:
Large datasets, VMs, and IDEs consume RAM. 16GB minimum for dev work, 32GB+ for heavy tasks.

🧹 PC Maintenance & Longevity

Keep your PC running smoothly for years!

Essential Maintenance Tasks:

  • Clean Dust Every 3-6 Months - Use compressed air, avoid vacuum (creates static)
  • Monitor Temperatures - CPU under 80°C, GPU under 85°C during load
  • Update Drivers Regularly - Especially GPU drivers for performance/security
  • Cable Management - Improve airflow and reduce dust accumulation
  • Replace Thermal Paste - Every 2-3 years or if temps increase
  • Check Fan Operation - Ensure all fans spin, replace if noisy
Advanced Maintenance:
• Update BIOS only if necessary (new CPU support, bug fixes)
• Use intake filters to reduce dust entry
• Maintain positive air pressure (more intake than exhaust)
• Keep ambient temperature reasonable (under 25°C ideal)
• Back up important data regularly
• Run disk cleanup and defrag (HDD) monthly
🛡️

Dust: The Silent Killer

Dust buildup can increase temps by 10-20°C and reduce component lifespan by years!

⚠️ When to Update BIOS:

Only update BIOS if:
✓ New CPU support needed
✓ Critical security patch
✓ Fixing specific hardware bug

Never update during storms or unstable power!
